# Pilot Partners — planning & discovery archive

Working artifacts from the pilot-partner (Gideon) engagement: call notes,
brainstorms, redesign plans, and clickable HTML prototypes. Kept for reference
as the consultant inquiry-to-case + client-portal work evolves.

## Structure

- **`call-notes/`** — raw notes from discovery calls (`gideon-2nd-call.md`, `gideon-3rd-call.md`).
- **`inquiry-to-case/`** — the inquiry → pre-case → case redesign plus the client-portal MVP.
  - `FINAL-PLAN-enquiry-to-case-and-client-portal.md` — the locked plan.
  - `inquiry-to-case-redesign-plan.html`, `consultant-inquiry-to-case-flow.html` — flow walkthroughs.
  - `client-portal-mvp.html` — client portal prototype.
- **`forms-builder/`** — questionnaire/forms-builder revamp: vision, brainstorm,
  implementation plan, and the diagram set (`*-01..06-*.png` + `*-flow.html`).
- **`e2e-prototypes/`** — three end-to-end approach prototypes for the agent
  pilot meeting (`approach-1` Atelier Pipeline / `approach-2` Pulse Copilot /
  `approach-3` Shared Case Room) plus `SPEC.md` and an `index.html` chooser.
